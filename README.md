# contributors

To install dependencies:

```bash
bun install
```

To run:

```bash
bun dev
```

The visit <http://localhost:3000/wxt-dev/wxt.svg>.

Optionally add a `.env` file with a token to increase rate limits:

```sh
GITHUB_TOKEN=your-token
```
